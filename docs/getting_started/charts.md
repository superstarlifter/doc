### Edit a chart

#### Change the chart type
  Click the hamburger menu in the upper left of the chart and select a chart type.
  
  <img src="../assets/charts_chart_type_1.png"  style="width:600px" class="border"></img>  

#### Quick edit
  Click the **Add Metric..** input to quickly add metrics to the chart (y-axis).
  
<img src="../assets/charts_addfield_1.png"  style="width:600px" class="border"></img>  

  Click the **Add Group by..** input to modify how chart values are grouped (x-axis).

<img src="../assets/charts_addfield_2.png"  style="width:600px" class="border"></img>  a
