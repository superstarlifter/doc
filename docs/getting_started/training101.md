## Bonus 3:  Use shortcuts

Press CONTROL to open the accelator key menu and select your desired key.

* A:  Admin
* B:  Dashboard & Grid
* C:  Chat
* D:  Dashboard
* F:  Find
* G:  Grid 
* I:  Insert Row
* V:  Filter Panel
* ?:  Documentation
