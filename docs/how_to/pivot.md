## How create a pivot
How-to video link - https://youtu.be/wkYBb-xqabo

Use a pivot to aggregate and group values by categories.

To create a pivot:

1.	Right click âž” **Set pivot**

<img src="../assets/pivot_0.png"  style="width:200px" class="border"></img>

2.  The pivot inserts category field as a **Pivot field** 

<img src="../assets/pivot_1.png"  style="width:400px" class="border"></img>

3.  Add, change and re-order **Pivot fields** by dragging

<img src="../assets/pivot_2.png"  style="width:600px" class="border"></img>

4.  The data defaults to **Group mode**.  This mode shows the groups.

<img src="../assets/pivot_3.png"  style="width:600px" class="border"></img>

5.  Expanding the **arrows** shows the granular detail.

<img src="../assets/pivot_4.png"  style="width:600px" class="border"></img>

6.  To change from **Group mode** to **Pivot mode**, select this icon:

<img src="../assets/pivot_5.png"  style="width:600px" class="border"></img>

7.  In **Pivot mode**, the second **Pivot field** is used as a column category.  See example below:

<img src="../assets/pivot_6.png"  style="width:600px" class="border"></img>

8.  To change back to **Group mode**, select the icon again:

<img src="../assets/pivot_7.png"  style="width:600px" class="border"></img>
